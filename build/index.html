Three distinct weights

Each icon in Streamline 3.0 has three unique weights. The differences are much more than just bumping up the stroke for each version too—each weight is a completely new icon. Check out the examples below.

Light

The classic Streamline style with bits. 1px stroke on a 24px grid.

Regular

A new minimal and geometric style. 1.5px stroke on a 24px grid.

Bold

 A new solid style akin to the latest iOS guidelines. 2px stroke on a 24px grid.

Crazy fast workflow

We’ve paid special attention to making Streamline faster and easier to use than ever before.

Drop these icons right into your workflow without skipping a beat, no matter which software or file format you prefer.

IconJar

One of the easiest ways to use the icons, if you're on a Mac. Open the .iconjar file and you have a beautifully organized list of every section and subsection. In addition, you can simply search for any icon by name and it will appear instantly. Drag and drop the vector icon straight from IconJar into your favorite design software. It just works.

Sketch

Each category of icon comes with its very own .sketch file. The icons have been pre-made into 24x24 symbols, meticulously named and organized for you. Works great with Sketch Runner and symbol overrides in general. Using these icons in Sketch has never been easier.

Figma

Figma opens Sketch files pretty flawlessly, but we went ahead and did the dirty work for you by converting the same Sketch icon category files into nice and tidy .fig files. You're welcome. Fun fact: Vincent and Matt used Figma to design this website.

Studio (Coming Soon)

We've ran some initial tests on converting our .sketch files into .studio files. The results are very promising and as soon as Studio is out of beta, our official files will be ready too.

Illustrator

Ahh, old faithful. The grandfather of all vector editing software. You can bet your bottom dollar we've got a bunch of .ai files for all of you Adobe fans out there.

SVG

SVGee whiz. We spent a lot of time converting all of these icons into clean code for all of the developers out there. All of the hard-coded HEX codes have been refactored to currentColor and the markup has been stripped down to the bare essentials.

Streamline icons in action

Streamline was created to enhance any project you're working on. Doesn't matter if it's a website, a mobile app, or a slide deck.

The light weights have a little more detail and can be scaled larger when more details are required. The regular weights are stripped down to very minimal shapes and work well for UI even when scaled down to 16px. The bold weights are nice and contrasty, perfect for iOS work, wayfinding, and more.

iOS Apps

The new bold Streamline style works amazingly well for iOS apps. The style was largely inspired by the latest iOS guidelines, which feature nice bold shapes for optimal legibility. The grid of icons at the top of this example are regular weight and the tab bar is using bold weight.

Websites

The website example above is using the regular weight icons in the left navigation. All of the icons are designed on a 24px grid, but the regular weight works well at smaller 16px sizes. One of our sister products, Streamline UX, is featured in the right panels.

Apple Watch

When you're designing interfaces, every single pixel matters. This is especially true when designing for tiny wearable screens. With Streamline you can be certain that each little vector shape has been fine-tuned for maximum value.
